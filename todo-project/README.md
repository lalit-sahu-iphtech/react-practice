# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some Oxlint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the Oxlint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and Oxlint's TypeScript related rules in your project.

# React Practice 🚀

This repository contains my React.js practice projects and concepts that I learned while exploring the fundamentals of React.

## 📚 Concepts Covered

* React Components
* JSX
* Props
* State (`useState`)
* Event Handling
* Dynamic Styling
* Conditional Rendering
* Component Reusability
* Building UI Components

## 🛠 Projects Included

### 🔢 Counter App

* Increment counter
* State management using `useState`

### ❤️ Like Button

* Toggle between liked and unliked states
* Font Awesome heart icon
* Conditional rendering

### 🛍 Product Card

* Display product details using reusable components
* Props-based data rendering
* Styled product cards

### 🎨 Dynamic Styling

* Apply styles dynamically based on state
* Change UI appearance using conditional CSS

### 🧩 Component Building

* Creating reusable React components
* Organizing UI into separate files

### 🖱 Event Handling

* Handling button clicks
* Using `onClick` events
* Updating state with event handlers

## 📂 Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Font Awesome

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd react-practice
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
